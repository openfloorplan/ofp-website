OpenFloorPlan

OpenFloorPlan is a set of open source JavaScript libraries that can be used to manipulate and interact with SVG-based floor plans in HTML5 or NodeJS.

For more information please see https://github.com/openfloorplan

If you are attending FOSS4G-NA, look for our table in the exhibitor area, and at the presentation Going Inside with FOSS4G - Developing Interactive Building Floor Plans in HTML5

Powered by Open Source Geospatial

Our goal is to bring open source geospatial tools inside the building footprint so they can be used to build a range of interior mapping and facility management solutions.
